Controls:
	Keyboard:
		WASD controls or arrow keys for player and menus
		Space or Z for jump and menu select
		LMB or X to shoot
		Esc to pause
	Gamepad(Xbox):
		Left stick and D-pad movement for player
		D-pad for menus
		A-button for jump and menu select
		X-button for shoot
		Start to pause
